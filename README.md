Учет и Анализ строительных материалов 

Android-приложение для учета строительных материалов, поставщиков и поставок.  

Основные возможности
  
- Аутентификация с ролями администратора и гостя  
- Управление материалами (добавление, редактирование, поиск)  
- Управление поставщиками  
- Управление поставками с цветовой индикацией статусов  
- Аналитика со статистикой  
- Настройки (темы, уведомления, экспорт CSV)  

Технологии  
- Kotlin, Jetpack Compose, Material Design 3  
- MVVM, Room, Coroutines, Flow  
- Минимальная версия Android: 7.0 (API 24)  

Тестовые пользователи  
-Администратор: admin / 1234  
-Гость: guest / guest123

Структура БД  
Таблицы: Materials, Suppliers, Deliveries, Users  

Установка  
1. Клонировать репозиторий  
2. Открыть в Android Studio  
3. Собрать и запустить  

Планы развития  
- Синхронизация с облаком  
- Расширенная аналитика  
- Сканирование штрих-кодов  
- Push-уведомления
